# postgres-stop
PostgreSQL Resource Center

### Introduction
The goal of this repository is to provide a resource center for learning PostgreSQL.

### Prerequisites
All of the work done here have been tested on Ubuntu 20.04 LTS and PostgreSQL 12.8.
